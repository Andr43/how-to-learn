# How to learn  
__________________
## В данном проекте описываются основные методики обучения, а также даются практические советы.  
* Основной **проблемой** при обучении является использование неправильных подходов и методик, из-за которых новый материал не усваивается и быстро забывается  
* **Виды техник обучения от Барбары Оакли:**  
1. Два вида внимания  
2. Recall  
3. Интерливинг  
4. Вопросы  
5. Иллюзия компетентности  
* Метод **Фейнмана** - эффективный метод обучения  
* и прочие методики и советы  
___________
## Данный проект является одностраничной страницей, в которой использованы следующие технологии:  
* Flexbox,  
* БЭМ,  
* Псевдоселекторы,  
* Iframe,  
* Keyframes,
* и другие технологии.
